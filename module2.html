02 / Hypertext Markup Language

Learning the language that structures every page on the Web.

Module Overview

HTML (Hypertext Markup Language) is the standard language for creating web pages. It describes the structure of a webpage using a series of elements represented by tags. This module teaches students how to write valid HTML documents from scratch.

Students will learn to use headings, paragraphs, links, images, lists, tables, and forms — the core building blocks of every website on the internet.

Topics Covered

  • Basic HTML Document Structure (<!DOCTYPE>, <html>, <head>, <body>)
  • Headings: <h1> through <h6>
  • Paragraphs, line breaks, and horizontal rules
  • Hyperlinks using the <a> tag
  • Images using the <img> tag
  • Ordered and Unordered Lists
  • HTML Tables — rows, columns, headers
  • HTML Forms — input fields, buttons, text areas
  • Semantic HTML5 Elements (<header>, <nav>, <main>, <footer>)

Sample HTML Code

Here is a basic HTML document structure studied in the lab:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8"/>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first web page.</p>
  </body>
</html>
