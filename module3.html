03 / Cascading Style Sheets

Controlling the look, feel, and layout of web pages with CSS.

Module Overview

CSS (Cascading Style Sheets) is the language used to style and lay out web pages. It controls colors, fonts, spacing, positioning, and responsive behavior. This module builds on the HTML skills from Module 2 and shows students how to transform plain HTML into visually appealing websites.

By the end of this module, students will be able to write and link CSS stylesheets, apply styles using selectors, and create basic responsive layouts.

Topics Covered

  • What is CSS and how it works with HTML
  • Three ways to apply CSS: Inline, Internal, External
  • CSS Selectors: element, class, id, descendant
  • Colors: named, hex, rgb, rgba
  • Typography: font-family, font-size, font-weight, text-align
  • The CSS Box Model: margin, border, padding, content
  • Background properties: color, image, repeat, position
  • Display and Positioning: block, inline, flex, relative, absolute
  • Introduction to Responsive Design with media queries

Sample CSS Code

A basic external stylesheet example practiced in the lab:

/* Styling the body */
body {
  font-family: Arial, sans-serif;
  background-color: #f0f4ff;
  color: #1e2d40;
}

/* Heading style */
h1 {
  color: #2e86ff;
  font-size: 2rem;
  text-align: center;
}
