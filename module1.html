01 / Web Programming Basics

Introduction to the Internet, the Web, and how they work together.

Module Overview

This module introduces the foundational concepts of the Internet and the World Wide Web. It covers how information travels across networks, how websites are hosted and served, and why web programming is an essential skill in today's digital world.

By the end of this module, students should understand the difference between the internet and the web, how a browser requests and receives web pages, and the roles of client and server in web communication.

Topics Covered

  • What is the Internet? History and Evolution
  • What is the World Wide Web (WWW)?
  • Difference between Internet and the Web
  • How Browsers and Web Servers Communicate
  • Client-Server Architecture
  • HTTP and HTTPS Protocols
  • Domain Names, URLs, and IP Addresses
  • Introduction to Web Programming Languages
  • Overview of Front-end vs Back-end Development
